# Frontend Agent

## Context

Vanilla HTML/CSS/JS in `static/`. No build step, no framework. "OpsCraft" dark theme.

## Key Files

| File | Purpose |
|------|---------|
| `login.html` | Auth page |
| `index.html` + `app.js` | Test console with WebRTC |
| `dashboard.html` + `dashboard.js` | Operations dashboard |

## Design System

- Dark theme (Discord/Linear inspired)
- CSS variables in each HTML file
- Glassmorphism effects
- No external CSS libraries
