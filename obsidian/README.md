# Obsidian Vault — Mantra Voice Agent

This vault is the **single source of truth** for the repository. Every AI agent or developer should start here to understand the project.

## How to Use This Vault

1. **Start at [[Home.md]]** for a project overview and navigation.
2. **Understand [[Architecture/Overview.md]]** before modifying code.
3. **Check [[Development/TODO.md]]** for in-progress and planned work.
4. **Update [[Development/Changelog.md]]** after every change.
5. **Persist knowledge** in [[Knowledge/]] and [[Context/]].

## Vault Structure

```
obsidian/
├── Home.md                     ← Entry point
├── README.md                   ← This file
├── Inbox/                      ← Unprocessed notes
├── Architecture/               ← System design
├── Features/                   ← Feature documentation
├── Development/                ← Sprint, TODO, bugs, changelog
├── Agents/                     ← AI agent instructions
├── Knowledge/                  ← Coding standards, conventions, glossary
├── Context/                    ← Project summary, stack, repo map
└── Templates/                  ← Note templates
```

## Convention

- All links use `[[WikiLink]]` format.
- Every code change must update relevant documentation.
- Architecture decisions are recorded in [[Architecture/Design Decisions.md]].
